Estrutura do Perfil

  • Avatar hero no topo: inicial grande com gradient + ícone de edição (lápis) + nome + badge de nível.
  • 2 grupos de campos editáveis: "sobre você" (apelido, idade, gênero) + "hábito" (tipo de aposta, há quanto tempo, gasto mensal).
  • Cada campo exibe o valor atual à direita + chevron indicando que é tappável pra editar.
  • Tipo de aposta exibe chips inline (esportivas, cassino) — visual compacto.
  • Todos os dados vêm do onboarding (P1-P6). Aqui o usuário pode corrigir sem refazer o fluxo.

Elementos especiais

  • Início da jornada — card com ícone de calendário + data em serif itálico. Marco simbólico, não editável.
  • Resetar contagem — zona de perigo (ícone lixeira em brand-deep). Em produção, pede confirmação com modal. Existe para honestidade — se a pessoa recaiu, pode resetar sem vergonha.
  • Bottom nav sem nenhuma tab ativa — o perfil não é uma tab, é acessado pelo avatar no header.
  • Botão voltar (← ) no header — volta pra tela anterior.
  • Nenhum dado sensível real é mostrado — tudo via apelido inventado, faixa etária, categorias gerais.
